Guía práctica

Cómo estructurar bien el inventario

Una guía corta para que el inventario de PC Gamer Margarita sea más fácil de buscar, vender, filtrar y mantener. No se trata de hacerlo perfecto desde el día uno: se trata de que cada producto diga lo suficiente para encontrarlo rápido y venderlo sin dudas.

Idea central

El nombre no debe hacer el trabajo de todas las columnas

Si el producto se llama Fuente Certificada 600 W Mlogis, la app puede mostrarlo, pero el equipo pierde claridad: no queda separado qué es categoría, marca, modelo o característica. Lo ideal es que el nombre sea legible y que la categoría, marca, modelo, condición y stock estén en campos separados.

Formato recomendado

Campo Qué debe tener Ejemplo
Categoría Tipo general del producto. Sirve para filtrar. PSU, GPU, CPU, MOUSE, TECLADO, SSD
Marca Fabricante o marca comercial. Mlogis, Redragon, Logitech, Kingston, Dell
Modelo Modelo o línea exacta si existe. G203, NV2, Zeus X, Optiplex 7050
Característica Lo que ayuda a distinguirlo: capacidad, watts, generación, color, tamaño. 600W, 1TB NVMe, i5 7ma, 8GB, blanco
Condición Nuevo o Refurbished. Importante para ventas e historial. Nuevo / Refurbished

Ejemplos de mejora

PSU

Fuente de poder

Antes: Certificada 600 W Mlogis
Mejor: Mlogis 600W Certificada

Campos sugeridos: Categoría PSU, Marca Mlogis, Modelo vacío si no existe, característica 600W certificada.

GPU

Tarjeta gráfica

Antes: Grafica RTX 2060 - 6 GB MSI Ventus Geforce
Mejor: MSI GeForce RTX 2060 Ventus 6GB

Campos sugeridos: Categoría GPU, Marca MSI, Modelo RTX 2060 Ventus, característica 6GB.

SSD

Almacenamiento

Antes: Almacenamiento SSD 1 TB Kingston NV2
Mejor: Kingston NV2 1TB NVMe

Campos sugeridos: Categoría SSD, Marca Kingston, Modelo NV2, característica 1TB NVMe.

PC

Equipo refurbished

Antes: Equipo PC Refurbished Dell i5 - 7ma. 8Gb
Mejor: Dell PC i5 7ma 8GB

Campos sugeridos: Categoría EQUIPO, Marca Dell, característica i5 7ma / 8GB, condición Refurbished.

Reglas simples para el día a día

No repetir la categoría en el nombre

Si ya existe la columna Categoría, no hace falta llamar el producto Mouse Logitech G203. Mejor Logitech G203.

Marca primero cuando se conozca

Ayuda a buscar rápido y evita listas confusas. Ejemplo: Redragon Zeus X, Logitech G203.

Características al final

Capacidad, color, generación, watts o tamaño deben ir después del modelo: Kingston NV2 1TB NVMe.

Condición separada

Usar el campo Nuevo / Refurbished. Si se deja dentro del nombre, luego cuesta filtrar y vender.

Stock por producto real

Si dos productos cambian por color, capacidad o generación, deben ser productos separados.

Precio pendiente no bloquea

Se puede cargar inventario con precio 0, pero esos productos deben revisarse antes de vender.

Cómo aprovechar mejor la app

Inventario

Para encontrar productos rápido

  • Usar categorías limpias: GPU, CPU, PSU, SSD, HDD, MOUSE, TECLADO.
  • Evitar nombres demasiado largos si los datos pueden ir en marca/modelo.
  • Corregir primero los productos de mayor rotación.
  • Revisar productos con stock 0 para decidir si siguen activos.
Ventas

Para vender sin errores

  • Completar precio base USD antes de vender.
  • Confirmar condición Nuevo o Refurbished.
  • Usar el carrito desde Inventario para descontar stock automáticamente.
  • Imprimir nota de entrega al finalizar la venta.
Plan recomendado

No limpiar todo de golpe

Lo más práctico es cargar todo el inventario actual y luego limpiar por prioridad: primero productos caros, luego productos de alta rotación, después productos con stock, y por último productos agotados o antiguos. Así la tienda puede empezar a usar la app sin esperar una limpieza perfecta.
